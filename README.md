## Overview

This project is based on a Frontend Mentor challenge where the objective is to recreate a clean, centered QR code card component that works well on both desktop and mobile screens. The layout includes:

- A centered white card with rounded corners
- A QR code image
- A heading and supporting paragraph aligned with proper typography and spacing
- A responsive layout that adjusts at smaller screen sizes using media queries

This project demonstrates foundational frontend development skills such as structuring HTML, applying CSS styling techniques, and implementing responsive design.
